# 📥 Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/vishakagangurde/AI-Automation-Workflows.git
```

---

## 2. Install n8n

```
npm install n8n -g
```

or use Docker.

---

## 3. Import Workflow

- Open n8n
- Import any `.json` workflow
- Configure credentials
- Run the workflow
